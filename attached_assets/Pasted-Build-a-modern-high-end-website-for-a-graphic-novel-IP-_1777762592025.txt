Build a modern, high-end website for a graphic novel IP called Heroes of Hellscape.

The site should feel cinematic, mysterious, and premium, inspired by the tone and structure of Strangeland Comics and Sigma Comics, but not copy them. It must feel like a next-gen entertainment IP, not a small indie comic page.

Tech Requirements
Framework: Next.js (App Router)
Styling: Tailwind CSS
Animations: Framer Motion
Fully responsive (mobile-first)
Dark mode default
Clean, scalable component structure
Optimized for performance and SEO
Core Experience Direction
Dark, immersive, cinematic UI
Strong typography, minimal clutter
Smooth transitions and subtle parallax
Focus on storytelling and worldbuilding
Feels closer to a premium game or film IP than a comic store
